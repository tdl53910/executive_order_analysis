# Executive Order AI Analysis

This project analyzes the semantic evolution of U.S. executive orders using natural language processing and machine learning techniques.

## Features

- **Automated Scraping**: Fetches executive orders from the Federal Register API
- **Text Preprocessing**: Cleans and normalizes text for analysis
- **Embedding Generation**: Uses state-of-the-art sentence transformers to create semantic embeddings
- **Dimensionality Reduction**: Applies PCA, t-SNE, and UMAP for visualization
- **Cluster Analysis**: Identifies patterns and groupings in executive orders
- **Semantic Drift**: Measures how presidential language has evolved over time
- **Interactive Visualizations**: Creates both static and interactive plots
